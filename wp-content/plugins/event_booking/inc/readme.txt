== Description ==

Event Booking is a simple Like button plugin for posts, pages, custom post types and WooCommerce products.


== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Event Booking'
3. Activate 'Event Booking' from your Plugins page.
4. Visit 'Settings > Event Booking' and change settings according to your website theme.

= From WordPress.org =

1. Download `event-booking` and extract the zip.
2. Upload `event-booking` directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Event Booking' plugin through the 'Plugins' menu in WordPress
